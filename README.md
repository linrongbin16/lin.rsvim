# lin.rsvim: Lin Rongbin's RSVIM Distribution

lin.rsvim is a highly configured [Rsvim](https://rsvim.github.io/) distribution integrated with tons of utilities for development.

> [!NOTE]
>
> If you are using [Neovim](https://neovim.io/), please take a look at [lin.nvim](https://github.com/linrongbin16/lin.nvim).

## Introduction

Aim to be out-of-box, IDE-like editing experience, performant, lightweight and friendly to most Rsvim users. Focus on and only on editing, no compiling/packaging/debugging.

This ultra config solves below issues:

- Duplicate installation on different OS and machines? - All done by one-line command (not on Windows for now), same behaviors on all platforms.
- Time-costing configurations? - All configs follow community best practice, vim tradition and most popular editors (just like [vscode](https://code.visualstudio.com/)).

## Get started

### MacOS/Linux

> [!NOTE]
>
> For MacOS please install [Xcode](https://developer.apple.com/xcode/) and [homebrew](https://brew.sh/) as pre-requirements.

```bash
git clone https://github.com/linrongbin16/lin.rsvim ~/.rsvim && cd ~/.rsvim && npm install
```

And that's all of it.
